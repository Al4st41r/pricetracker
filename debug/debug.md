# Debugging Tools

This file documents the debugging tools and scripts available in this project.

## Playwright Website Check

-   **File:** `check_website.js`
-   **Description:** A Playwright script that launches a headless browser, navigates to the website, and takes a screenshot.
-   **Usage:**
    ```bash
    cd debug
    node check_website.js
    ```
-   **Output:** A `screenshot.png` file will be created in the `debug` folder.
